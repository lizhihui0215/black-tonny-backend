# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-21

### Added
- Initial project structure for Black Tonny Backend.
- FastAPI application core with routing and security configuration.
- Database integration (MySQL/SQLAlchemy) and initialization scripts.
- API endpoints for cost snapshots, jobs, manifest, and status.
- Services for analysis loading, cost management, and job handling.
- Docker and Nginx deployment configurations.
- Sample data for dashboard, details, and relationships.
- Unit tests for application core.
- Environment configuration templates.
